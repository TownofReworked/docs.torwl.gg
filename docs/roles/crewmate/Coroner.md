### _Freeze bodies so they stay!_

The **Coroner** can freeze any body before a meeting. If a body is frozen, it gets a frozen overlay, and stays on the map, even after a meeting has ended. This body can only be reported **ONCE**.

### Abilities

| Button | Ability | Description | Type |
| ------ | ------- | :---------: | :--: |
| <img src={useBaseUrl('/img/Buttons/Freeze.png')} alt="Button" style={{width: 50}} /> | **Freeze** | Freeze a dead body so that it stays. | Body Interaction |

### Options

| Option | Description | Type | Default |  Range   |
| ------ | :---------: | :--: | :-----: | :------: |
| Freeze Cooldown | Range where arrows will be colored. | Seconds | 15s | 0s - 50s  |
| Freeze Uses | Amount of uses for the Freeze button. | Count | 0 | 0 - 10 |

### Strategy Guide

As the Coroner, you can freeze bodies so they stay forever. There really is not anything else, just that.
